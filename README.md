# 联盟视界
英雄联盟国服战绩助手工具 | LoL Data Companion

> 实时查询战绩、卡片展示、选人提醒，让你的峡谷之旅更智能！

---

## 功能特色

### 1. 实时战绩查询
- 软件启动后自动查询当前玩家的战绩
  - 包括胜负、KDA、出装、符文配置
  - 显示本场伤害、经济、参团率及排名（与 WeGame 数据一致）

### 2. 数据筛选
- 支持匹配、单双排、灵活模式的战绩筛选
- 可按需求自定义查询范围

### 3. 选人阶段辅助
- 游戏开始选人阶段自动跳转到战绩界面
- 自动查询我方与敌方近期 20 场战绩
- 显示是否组队玩家，组队玩家名称颜色统一，便于辨识

### 4. 预选英雄配置
- 可添加常用英雄到预选区并自动保存
- 游戏时自动切换排名靠前的英雄
- 支持匹配、排位、大乱斗模式

### 5. 消息提醒功能（v1.0.7+）
- 选人窗口战绩消息发送，可自定义消息
- 游戏内战绩消息发送，可自定义消息

### 6. 自动更新
- 支持版本自动检测与更新（v1.0.6+）
